# UI Design Rules

These rules ensure consistent design across the Telecom Marketplace.

---

# Design Philosophy

Clean SaaS interface

Minimal UI

Fast loading

No unnecessary components.

---

# Layout Rules

Landing pages must include:

Header  
Main section  
Footer

Dashboard pages must include:

Sidebar or header navigation  
Content area

---

# Typography

Primary font

Geist Sans

Headings

bold  
large  
clear hierarchy

---

# Colors

Primary

black

Secondary

gray scale

Background

white

Avoid excessive colors.

---

# Buttons

Primary button

black background  
white text

Secondary button

border only

---

# Landing Page Structure

Landing page should include:

Header

logo  
login  
signup

Hero section

logo or product visual  
short description  
CTA

CTA buttons

Get Started  
Login

Footer

© LEOTEOR LLC  
Terms  
Privacy  
Contact

---

# Dashboard UI

Dashboards should prioritize:

clarity  
speed  
data visibility

Avoid decorative elements.

---

# Forms

Forms should be simple.

Group fields logically.

Example

Company info  
Insurance  
Certifications

---

# Responsive Design

All pages must work on:

desktop  
tablet  
mobile

Mobile layout must not break forms or navigation.
